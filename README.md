# PokemonFireRedBattleAI

An AI that takes control of battles (single battles for now) in the game Pokemon FireRed.

Requirements:
- Emulator for Pokemon FireRed
- Tesseract OCR
- Node.js and npm

Packages (pip install):
- re
- cv2
- mss
- time
- math
- random
- ctypes
- subprocess
- numpy
- pandas
- difflib
- pypokedex
- pytesseract
- pydirectinput
- PIL
- win32gui

Progress:
AI can currently do single battles, but there is no double battle support for this AI (yet). <br />
